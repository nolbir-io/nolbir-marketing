## Status Options

- Planned
- Running

## Rules

- No experiment starts without one primary metric and one threshold.
- Every completed test must end with one decision: scale, iterate, or stop.
- Reference decision updates in `docs/playbook/open-questions-register.md` when assumptions shift.
