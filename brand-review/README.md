# brand-review

Process and rubric for checking that any piece of Mamurify content sounds like Mamurify before it ships.

## Purpose

Catch off-brand language, off-message claims, and off-audience framing before content goes out. Depends on `brand-voice/` for the source of truth — this folder owns the *check*, not the *standard*.

## Key questions to answer

- What's the rubric — which dimensions are pass/fail (voice, claims, language correctness, legal/regulatory, accessibility)?
- Who reviews what? Founder for strategic claims, marketing lead for voice, native-speaker review for UZ/RU copy?
- What's the SLA — how long does a piece sit in review?
- How do we record decisions and feed corrections back into `brand-voice/`?
