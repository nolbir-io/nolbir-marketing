# Language Matrix

Guidance for language selection by channel and asset type.

## Default Policy (to validate in open questions)

- **Primary:** Uzbek
- **Secondary:** Russian
- **Optional by audience:** English

## Channel Matrix

| Channel | Primary language | Secondary language | Notes |
|---|---|---|---|
| Landing pages | Uzbek | Russian | Keep core value proposition mirrored |
| Sales decks | Uzbek | Russian | Add English only for partner/investor use |
| Social short-form | Uzbek | Russian | Adapt copy to channel norms |
| Email nurture | Uzbek | Russian | Segment by audience preference |
| Partner collateral | Russian | Uzbek | Tailor by partner profile |

## QA Workflow

1. Draft in source language.
2. Translate/adapt for second language.
3. Native reviewer validates meaning and tone.
4. Brand reviewer confirms consistency.
