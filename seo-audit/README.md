# seo-audit

Search visibility for NOLBIR across the queries Uzbek merchants and adjacent audiences actually type.

## Purpose

Map the keyword universe in UZ / RU / EN, audit the current site against it, and identify the technical and content gaps worth closing. Feeds opportunity briefs into `content-creation/` and tracks rankings in `performance-analytics/`.

## Key questions to answer

- What does an Uzbek merchant search when they hit the problem NOLBIR solves — in Uzbek (Latin? Cyrillic?), in Russian, in English?
- Who currently ranks for those queries? What does the SERP look like (marketplaces, news, competitor blogs, Telegram channels)?
- What's the technical state of the existing NOLBIR site — crawlability, speed, hreflang, structured data, mobile?
- What's the on-page state — does each priority page have a clear primary keyword, intent match, internal links?
- Where are the content gaps where ranking is realistic in 6–12 months?
